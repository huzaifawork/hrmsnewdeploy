🔖 HRMS Responsive Testing Bookmarklet

📋 How to Create the Bookmarklet:

  1. Copy the code below
  2. Create a new bookmark in your browser
  3. Paste the code as the URL
  4. Name it "HRMS Responsive Test"
  5. Click the bookmark on any page to test

🔗 Bookmarklet Code:

javascript:(function(){ var d=document,s=d.createElement('div'); s.innerHTML='

📱 Responsive Test

'; d.body.appendChild(s); var w=window.innerWidth,h=window.innerHeight,r=d.getElementById('testResults'); r.innerHTML='Screen: '+w+'x'+h+'
'; r.innerHTML+='Device: '+(w<=480?'📱 Mobile':w<=768?'📱 Large Mobile':w<=1024?'💻 Tablet':'🖥️ Desktop')+'
'; r.innerHTML+='Scroll: '+(d.body.scrollWidth>w?'❌ Horizontal':'✅ None')+'
'; var btns=d.querySelectorAll('button,a'),small=0; btns.forEach(function(b){var rect=b.getBoundingClientRect();if(rect.width<44||rect.height<44)small++;}); r.innerHTML+='Touch Targets: '+(small===0?'✅ Good':'⚠️ '+small+' small')+'
'; r.innerHTML+='Background: '+(getComputedStyle(d.body).backgroundColor==='rgb(255, 255, 255)'?'✅ White':'⚠️ Not white'); })();

✨ What the Bookmarklet Does:

  • 🔍 Shows current screen size and device type
  • 📏 Checks for horizontal scrolling issues
  • 👆 Counts small touch targets (buttons < 44px)
  • 🎨 Verifies white background theme
  • 📊 Displays results in a popup overlay

🎯 Quick Manual Tests:

For each page, check:

  • ✅ White background
  • ✅ Black text
  • ✅ Mobile navigation works
  • ✅ No horizontal scrolling
  • ✅ Buttons are touch-friendly
  • ✅ Forms work on mobile
  • ✅ Images scale properly
